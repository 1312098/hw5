** Hướng dẫn chạy chương trình **
- Để chạy chương trình đầu tiền cần cài nodejs
- Sau đó mở cmd lên và di chuyển tới thư mục source
- Chạy lệnh đầu tiên npm install để cài package express dùng để host lên.
- Sau đó chạy lệnh node app.js
- Mở trình duyệt và truy cập vào localhost:3000
